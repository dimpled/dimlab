CONFIGURATION
-------------

### Database

Edit the file `config/db.php` with real data, for example:

```php
return [
    'class' => 'yii\db\Connection',
    'dsn' => 'mysql:host=localhost;dbname=dimlab',
    'username' => 'xxx',
    'password' => 'xxx',
    'charset' => 'utf8',
];
```
